NBA Park Arduino Library

R&D on projects | Present

Worked as a programmer on projects for Arduino-controlled attractions throughout the theme park, building experiences focusing on game feedback and maintainability. Here are the attractions where the library was used:

R-Battle

Tabletop basketball game using remotely controlled tanks, with matches up to 4 players. I programmed the game logic, addressable LEDs, IR sensors on the "baskets" to count each scored point, the content on the TFT display, and a button for interaction and configuration.

MVP Competion

Pop-a-shot-like game where players compete for the highest score, having three hoops to make shots, where a projection shows which one is active for scoring. I programmed the game and integration logic with the projection mapping software, using OSC protocol for information exchange between the microcontroller and the computer running the projections, as well as US sensors attached to the hoops to count scored points.

C++ | Memory Management | PlatformIO | Arduino | OOP | UX | Communication Protocols | Embedded | Electronics

IR with Inverted Index & ATM SIM

Google's C++ Education Course | December 2024

The course's final project is to build a search engine by creating an Inverted Index data structure from text documents, providing a command-line UI for query input and search result display, while the previous was an ATM Simulator, both illustrating I/O and OO design in C++ and project structuring. By requiring multiple files, write a hash table from scratch and perform dynamic memory allocation, those projects helped me better understand the C/C++ build process and memory management as a whole, while also giving me plenty of room to explore tools like Make and GDB, which I was curious to learn at the time.

C++ | GDB | Memory Management | Makefiles | Information Retrieval | OOP

Pygame Ghouls 'n Ghosts

CS50P's Final Project | July 2024

A 2D arcade-style game inspired by Super Ghouls 'n Ghosts, featuring a leaderboard system that stores the top ten highest scores and the names of the players who achieved them. This project pushed me into prototyping and implementing game mechanics, such as score and stat tracking systems, collisions, enemy behaviors, dynamic item and weapon systems, player leveling, game balancing, customizable controls, and UI. The game can be downloaded on Itch.io.

Python | Pygame | Game Programming Logic | UX | OOP
